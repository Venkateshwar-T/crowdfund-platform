# **App Name**: Welcome Canvas

## Core Features:

- Static Homepage Display: Render a single, static page displaying the text 'Hello World'.
- Centered Content: Ensure the 'Hello World' text is prominently centered both horizontally and vertically on the page.

## Style Guidelines:

- Main text color: Teal (#78999D) for the 'Hello World' greeting, providing a clean and calming presence.
- Background palette: A clean and serene gradient or layered background featuring Aqua (#9CC2C6), Light Green (#E4EFEA), and White (#FAF8ED).
- Headline and body font: 'Inter' (sans-serif) for a modern, objective, and clear presentation of the 'Hello World' message.
- The 'Hello World' text is centrally aligned to provide immediate focus and visual balance.
